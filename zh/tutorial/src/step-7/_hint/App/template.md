| 商品 | 价格 | 数量  | 备注 |
|----|---:|:---:|:---|
| 苹果 | $1 | 5 : |    |
| 香蕉 | $2 |  3  |    |

在下方试试创建本节文档描述中提到的表格。（此编辑器支持复制粘贴表格）

| 功能   | Markdown 语法    |
|------|----------------|
| 加粗   | `**文本**`       |
| 倾斜   | `_文本_`         |
| 删除线  | `~~文本~~`       |
| 加粗倾斜 | `**_文本_**`     |
| 全部格式 | `**_~~文本~~_**` |
