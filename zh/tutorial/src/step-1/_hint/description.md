# Markdown 中的标题

Markdown 中标题以`# `开头，一个`#`号再加一个空格，后面加上文字，就能让这一段变成一级标题。

虽然 Markdown 的标题一共有六级。但是不建议经常用四级以上标题。

| 功能   | 快捷键            | Markdown 语法 |
|------|----------------|-------------|
| 一级标题 | Ctrl + Alt + 1 | `#`         |
| 二级标题 | Ctrl + Alt + 2 | `##`        |
| 三级标题 | Ctrl + Alt + 3 | `###`       |
| 四级标题 | Ctrl + Alt + 4 | `####`      |
| 五级标题 | Ctrl + Alt + 5 | `#####`     |
| 六级标题 | Ctrl + Alt + 6 | `######`    |
