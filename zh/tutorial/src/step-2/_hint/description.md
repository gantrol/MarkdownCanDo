# Markdown 中的文本格式化

Markdown 允许你使用简单的符号来格式化文本。

- 加粗：用两个星号`**`包围文本。
- 倾斜：用一个下划线`_`包围文本。
- 删除线：用两个波浪线`~~`包围文本。

你可以组合使用这些格式标记来给文本添加多重格式，例如同时加粗、倾斜和添加删除线。

| 功能   | Markdown 语法    |
|------|----------------|
| 加粗   | `**文本**`       |
| 倾斜   | `_文本_`         |
| 删除线  | `~~文本~~`       |
| 加粗倾斜 | `**_文本_**`     |
| 全部格式 | `**_~~文本~~_**` |

如果你手动键入来加粗，那么可能也注意到`*文本*`也可以用来*倾斜*。

当然，中文不太建议倾斜，会*很难看*。
